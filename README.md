## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/lucaliebenberg/go-hotel-reservation-json-API.git
   ```

2. Build the project:
  ```bash
  go build
  ```

3. Run the project:
  ```bash
  ./project
  ```
